## Tecnologias
- Java 17
- Spring Boot 2.6.3
- Maven
- Lombok
- MySQL

## Como rodar
### Configuração do banco de dados
Crie um banco de dados MySQL e configure o arquivo `src/main/resources/application.properties`.

### Executar
```
.\mvnw spring-boot:run
```

A aplicação estará disponível em `http://localhost:8080`.
